valueFiles

abstract val valueFiles: ConfigurableFileCollection (source)

A collection of YAML files containing values. These will be applied to all releases when this target is active.

Corresponds to the --values CLI option.

Not to be confused with fileValues, which contains entries whose values are the contents of files.
