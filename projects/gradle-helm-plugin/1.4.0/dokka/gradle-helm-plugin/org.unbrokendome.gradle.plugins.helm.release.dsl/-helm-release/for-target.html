forTarget

open fun forTarget(target: String, action: Action<TargetSpecific>): Unit (source)

Adds a target-specific configuration action for the given release target.

The target name can be prefixed with an exclamation mark (e.g. !target) to apply the configuration for any but the given target.

Parameters

target - the name of the release target

action - the target-specific configuration to apply
