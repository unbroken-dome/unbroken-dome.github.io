HELM_CHART_PUBLISHING_CONVENTION_NAME

const val HELM_CHART_PUBLISHING_CONVENTION_NAME: String (source)

The name of the publishing convention object installed on each chart.

Note that since convention properties are accessible directly on the object, the name of the convention does not really matter except for uniqueness.
