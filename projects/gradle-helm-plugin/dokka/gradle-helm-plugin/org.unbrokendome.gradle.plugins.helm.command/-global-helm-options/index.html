GlobalHelmOptions

interface GlobalHelmOptions

Holds options that apply to all Helm commands.

Properties

debug

 abstract val debug: Provider<Boolean>

Indicates whether to use the verbose output (--debug flag) when invoking commands.

executable

 abstract val executable: Provider<String>

The name or path of the Helm executable. The PATH variable is taken into account, so this can just be "helm" if the Helm client is installed in a suitable location.

extraArgs

 abstract val extraArgs: ListProperty<String>

Additional command-line arguments to pass to the Helm CLI.

home

 abstract val home: Provider<Directory>

The path of the Helm home directory. If this provider has a value, it will be used to set the HELM_HOME environment variable for each Helm invocation.
