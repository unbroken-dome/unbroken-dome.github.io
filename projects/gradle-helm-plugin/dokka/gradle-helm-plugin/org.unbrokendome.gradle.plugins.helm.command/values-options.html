valuesOptions

fun HelmExecSpec.valuesOptions(values: Provider<Map<String, Any>>, valueFiles: FileCollection): Unit

Adds common options for commands that take values (install, upgrade, lint).

This will add --set, --set-string and --values based on the supplied properties.

Parameters

values - a Provider of a Map containing directly specified values. Entries with String values will be added using --set-string; other types of values will be added using --set.

valueFiles - a FileCollection of YAML files containing additional values to pass to the command.
