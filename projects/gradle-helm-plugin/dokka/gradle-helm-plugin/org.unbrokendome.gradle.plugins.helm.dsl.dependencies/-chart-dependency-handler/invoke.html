invoke

open operator fun String.invoke(chart: String = "main", project: String? = null): Unit

Adds a dependency on another chart.

This variant is intended for Kotlin DSL support, allowing us to declare a chart dependency like this:

dependencies {
    "foo"(project = ":bar", chart = "main")
}

Parameters

chart - the name of the chart inside the Gradle project. Defaults to "main" if not specified.

project - the path to the project that contains the chart. If null, the dependency will be on a chart inside the same project as the dependent chart.

Receiver
the name of the chart in the dependent chart's requirements. This should either match the name or alias field inside the dependent chart's requirements.yaml.
