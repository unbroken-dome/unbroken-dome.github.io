abstract fun add(name: String, chart: String = "main", project: String? = null): Unit

Adds a dependency on another chart.

Parameters

name - the name of the chart in the dependent chart's requirements. This should either match the name or alias field inside the dependent chart's requirements.yaml.

chart - the name of the chart inside the Gradle project. Defaults to "main" if not specified.

project - the path to the project that contains the chart. If null, the dependency will be on a chart inside the same project as the dependent chart.

open fun add(notation: Map<*, *>): Unit

Adds a dependency on another chart, using a map notation.

This variant is intended for Groovy DSL support, allowing us to declare a chart dependency like this:

dependencies {
    add(name: 'foo', project: ':bar', chart: 'main')
}

The following keys are supported in the notation map parameter:

  • name: the name of the chart in the dependent chart's requirements
  • chart: the name of the chart inside the Gradle project
  • project: the path to the project that contains the chart

Parameters

notation - a Map containing the dependency properties
