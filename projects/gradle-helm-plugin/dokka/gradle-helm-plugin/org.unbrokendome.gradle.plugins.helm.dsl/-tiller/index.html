Tiller

interface Tiller

Defines options for setting up Tiller on a remote Kubernetes cluster.

Properties

forceUpgrade

 abstract val forceUpgrade: Property<Boolean>

If true, force upgrade of Tiller to the current Helm version.

historyMax

 abstract val historyMax: Property<Int>

Limit the maximum number of revisions saved per release. Use 0 for no limit.

image

 abstract val image: Property<String>

Override Tiller image.

install

 abstract val install: Property<Boolean>

Whether to install Tiller on the cluster.

namespace

 abstract val namespace: Property<String>

Namespace of Tiller (default "kube-system")

replicas

 abstract val replicas: Property<Int>

Amount of Tiller instances to run on the cluster.

serviceAccount

 abstract val serviceAccount: Property<String>

Name of service account.

upgrade

 abstract val upgrade: Property<Boolean>

Upgrade if Tiller is already installed.

wait

 abstract val wait: Property<Boolean>

If true, block until Tiller is running and ready to receive requests.
