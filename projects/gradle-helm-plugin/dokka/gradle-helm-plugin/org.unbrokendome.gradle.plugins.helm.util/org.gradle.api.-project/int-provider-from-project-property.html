intProviderFromProjectProperty

fun Project.intProviderFromProjectProperty(propertyName: String): Provider<Int>

Returns a Provider that provides the given project property if it is available, interpreting its value as an integer.

The project property is resolved as per Project.property.

If the type of the property value is a Number, the provider will return its integer value; if it is a String, the provider will convert it to an integer. The provider will throw an exception if the property is not a Number or String.

Parameters

propertyName - the name of the property

Receiver
the Gradle Project

Return
a Provider that returns the project property value if it exists, or is empty if the property does not exist
