HelmUpdateDependencies()

Updates the chart dependencies from the requirements.yaml file, and creates a requirements.lock file that fixes the versions of chart dependencies. Corresponds to the helm dependency update CLI command.

This task will be skipped with NO_SOURCE if the chart does not a requirements.yaml file.
