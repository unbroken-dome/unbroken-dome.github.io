HelmBuildOrUpdateDependencies()

Builds or updates chart dependencies. This is a combination of the helm dependency update and helm dependency build CLI commands, which executes either update or build according to the following logic:

  • if the requirements.yaml file does not exist, deletes the requirements.lock file (if present) and does not execute either of the CLI commands.
  • if requirements.yaml exists but requirements.lock does not, executes helm dependency update.

  • if both requirements.yaml and requirements.lock exist:

    • if requirements.yaml is newer than the lock file, executes helm dependency update.
    • if the requirements.lock is newer than requirements.yaml, executes helm dependency build.
