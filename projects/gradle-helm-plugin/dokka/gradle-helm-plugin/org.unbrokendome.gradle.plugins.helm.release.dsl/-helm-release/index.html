HelmRelease

interface HelmRelease : Named

Represents a release to be installed into or uninstalled from a Kubernetes cluster.

Properties

If true, will execute the release atomically.

Names of other releases that this release depends on.

fileValues

 abstract val fileValues: MapProperty<String, Any>

Values read from the contents of files, to be used for the release.

keepHistoryOnUninstall

 abstract val keepHistoryOnUninstall: Property<Boolean>

If true, the associated helmUninstall task will retain the release history (using the --keep-history flag).

namespace

 abstract val namespace: Property<String>

The namespace to install the release into.

releaseName

 abstract val releaseName: Property<String>

The release name to use for any Helm CLI command. Defaults to the name of the release in the DSL.

replace

 abstract val replace: Property<Boolean>

If true, the associated helmInstall task will replace an existing release with the same name (using helm install --replace instead of helm upgrade --install).

repository

 abstract val repository: Property<URI>

Chart repository URL where to locate the requested chart.

valueFiles

 abstract val valueFiles: ConfigurableFileCollection

A collection of YAML files containing values for this release.

values

 abstract val values: MapProperty<String, Any>

Values to be used for the release.

version

 abstract val version: Property<String>

Specify the exact chart version to install. If this is not specified, the latest version is installed.

wait

 abstract val wait: Property<Boolean>

If true, will wait until all Pods, PVCs, Services, and minimum number of Pods of a Deployment are in a ready state before marking the release as successful.
