ConfigurableHelmReleaseTestOptions

interface ConfigurableHelmReleaseTestOptions : HelmReleaseTestOptions (source)

Configures remote testing for a release using helm test.

Properties

enabled

Whether testing is enabled for this release (default true).

abstract val enabled: Property<Boolean>

showLogs

If true, dump the logs from test pods (this runs after all tests are complete, but before any cleanup).

abstract val showLogs: Property<Boolean>

timeout

The timeout to use when testing this release using helm test. If not set, defaults to the same value as remoteTimeout for the release.

abstract val timeout: Property<Duration>
