HelmReleaseTestOptions

interface HelmReleaseTestOptions (source)

Remote testing options for a release using helm test.

Properties

enabled

Whether testing is enabled for this release (default true).

abstract val enabled: Provider<Boolean>

showLogs

If true, dump the logs from test pods (this runs after all tests are complete, but before any cleanup).

abstract val showLogs: Provider<Boolean>

timeout

The timeout to use when testing this release using helm test. If not set, defaults to the same value as remoteTimeout for the release.

abstract val timeout: Provider<Duration>
