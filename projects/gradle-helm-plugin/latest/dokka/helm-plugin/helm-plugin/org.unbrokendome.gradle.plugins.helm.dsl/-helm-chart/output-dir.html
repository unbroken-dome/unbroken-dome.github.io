outputDir

open val outputDir: Provider<Directory>
Content copied to clipboard

The directory where the (exploded) chart files will be written.

This is read-only (a Provider, not a Property) because Helm demands that the chart directory has the same name as the chart. To change the base output directory, set the baseOutputDir property to a different value.
