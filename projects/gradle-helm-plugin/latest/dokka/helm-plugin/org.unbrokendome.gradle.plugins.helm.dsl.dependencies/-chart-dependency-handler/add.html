Parameters

project

the path to the project that contains the chart. If null, the dependency will be on a chart inside the same project as the dependent chart.

open fun add(notation: Map<*, *>)

Adds a dependency on another chart, using a map notation.

This variant is intended for Groovy DSL support, allowing us to declare a chart dependency like this:

dependencies {
    add(name: 'foo', project: ':bar', chart: 'main')
}
