xjcClasspathConfigurationName

val xjcClasspathConfigurationName: String (source)

The name of the XJC classpath configuration for this source set. This configuration should be used for adding plugins to the XJC generation step.

Any dependencies added to the XJC classpath configuration will be passed to XJC using the -classpath CLI option.
