extraArgs

abstract val extraArgs: ListProperty<String> (source)

Additional arguments to be passed to XJC for this source set.

This property will be initialized with any extraArgs specified in the global xjc block and any xjcExtraArgs specified in the source set. If you would like to preserve those, make sure to add to this list instead of replacing its contents.
