extension

abstract val extension: Property<Boolean> (source)

If true, enable JAXB vendor extensions.

The default is false, but extension mode be enabled automatically when any -X argument is present in the XjcGenerate.extraArgs list.

Corresponds to the -extension CLI parameter.
