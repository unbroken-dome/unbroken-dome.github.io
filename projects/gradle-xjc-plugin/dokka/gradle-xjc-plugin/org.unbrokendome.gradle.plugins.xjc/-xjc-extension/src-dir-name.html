srcDirName

abstract val srcDirName: Property<String> (source)

The conventional name of the XJC source directories.

The default is "schema".

For every source set, the corresponding XJC generator will look for schemas (*.xsd), binding files (*.xjb) and URL files (*.url) in a directory src/<source-set-name>/<srcDirName>. For example, if this property is set to "schema", XJC generation for the main source set will use sources from src/main/schema.

Change this property to change the conventional name for all source sets.
