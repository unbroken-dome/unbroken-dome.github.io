xjcGenerateEpisode

val SourceSet.xjcGenerateEpisode: Property<Boolean> (source)

If true, instructs XJC to generate an episode file at META-INF/sun-jaxb.episode. The generated episode file will then be included in the resources of this source set.

The default is false.
